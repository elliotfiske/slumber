Futuristic City

By Edgard Arroliga

Description

This project attempts to make a aesthetically pleasing scene of complicated .obj files and texture found online. Using these free resources, I was able to create a small futuristic city with some animated models. The city is lighted using a full phong shader and directional lighting to simulate midnight and textured to have a chrome floor and surrounded by a city skyline. In the demo, you can also walk around the world using the camera. The camera is restricted to only x-axis and z-axis movement in order to simulate a walking human. I was unable to add collision detection that worked properly for the robots and the buildings.
